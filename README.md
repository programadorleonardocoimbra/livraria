# 📚 Sistema de Gerenciamento de Biblioteca
Um sistema robusto desenvolvido em Java utilizando os princípios da Programação Orientada a Objetos (POO).
Este projeto demonstra os quatro pilares da POO — abstração, encapsulamento, herança e polimorfismo — aplicados em um caso real de gerenciamento de biblioteca.

## 🚀 Funcionalidades
Cadastro de clientes e funcionários.

- Controle de livros disponíveis para empréstimo.

- Registro de empréstimos e devoluções.

- Estrutura modular e expansível para novos tipos de itens (DVDs, revistas, etc.).

### 🛠️ Tecnologias Utilizadas
- Java 17+

### Paradigma POO

Estrutura modular para fácil manutenção e expansão
